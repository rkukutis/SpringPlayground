# SpringPlayground
A place to learn and test out various Spring framework features while building a simple API. Inspired by TECHIN Springine team's timetable app.
## Backend Structure
+ Configurations
+ Exceptions
+ Domains
  + Model
  + Controller
  + DTO
  + Mapper
  + Service
  + Repository
+ Validation units for each domain
## Built with
+ Logging: SLF4J
+ DB: in-memory (H2) -> PostgresSQL
+ Spring Data JPA
+ Hibernate validator
+ Spring Web
+ Lombok
+ Springfox
+ openPDF
+ JUnit
+ Flyway
